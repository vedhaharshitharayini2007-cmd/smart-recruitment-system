# Smart Recruitment System

## Overview
Smart Recruitment System is a web-based platform that streamlines the hiring process by connecting recruiters and job seekers. It enables job posting, candidate registration, application tracking, and recruitment management through an intuitive and user-friendly interface.

## Technologies Used
- React.js
- JavaScript
- HTML
- CSS
- Vite

## Features
- Job Posting and Management
- Candidate Registration
- Application Tracking
- Recruiter Dashboard
- Responsive User Interface
